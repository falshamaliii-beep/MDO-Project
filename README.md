## Summary
# MDO-Project
 project exploring Microsoft Defender for M365 threat detection and response capabilities
Microsoft Defender for Office 365 – Threat Detection & Email Security Analyzer

This project explores how Microsoft Defender for Office 365 detects, analyzes, and mitigates email‑based threats such as phishing, malware, and malicious URLs. It demonstrates how AI‑driven signals, Safe Links, Safe Attachments, and real‑time threat intelligence can be used to improve organizational security posture. The goal is to help security teams visualize risks, understand attack patterns, and automate response actions.

MDO Security Insights & Automated Threat Response

This project integrates Microsoft Defender for Office 365 APIs to collect threat data, classify email risks, and visualize attack trends. It uses machine‑learning‑based scoring to identify high‑risk messages and demonstrates automated remediation workflows to reduce response time and strengthen email security.
